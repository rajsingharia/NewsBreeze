# NewsBreeze
#Assignment For GreedyGame</br>
#features:</br>
-Breaking News.</br>
-Saving for Offline(using Room DB).</br>
-Caching mechanism(using Room DB).</br>
-Swip to delete saved articles.</br>
-Refresh for latest news articles.</br>
-when offline shows banner for offline and when again online -> automatic request for latest news articles.</br>
-Sort by date</br>
